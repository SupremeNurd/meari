# 📦 Meari (Publishing)

메아리(Meari) 프로젝트의 **퍼블리싱 1차 개발 소스 코드**입니다.  
이 프로젝트는 **Gulp.js**를 기반으로 빌드되며, HTML, CSS, JS 등의 소스 파일을 `dist` 폴더에 최종 산출물로 생성합니다.

백엔드 개발 시에는 **`dist` 폴더의 빌드 결과물**을 연동하여 사용하거나,  
API 연동을 위해 **`src` 폴더의 소스 코드**를 수정한 후 다시 빌드하여 테스트할 수 있습니다.

---

## 🛠️ 시작하기

### 1. 필수 환경

- **Node.js** (LTS 버전 권장)
- **npm** (Node.js 설치 시 자동 포함)

---

### 2. 의존성 설치

프로젝트를 클론(clone) 받은 후, 루트 디렉터리에서 아래 명령어를 실행하여 필요한 모든 패키지를 설치합니다.

```bash
npm install


사용 가능한 명령어
1. 개발 서버 실행 (실시간 변경 감지) : 소스 코드 변경 시 자동으로 빌드하고 브라우저를 새로고침합니다.

gulp

2. 프로덕션(배포용) 빌드 : 서버 배포를 위한 최종 산출물을 dist 폴더에 생성합니다.

gulp build
